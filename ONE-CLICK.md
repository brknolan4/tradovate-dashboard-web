# One-click launch

## Start

Run:

```bash
bash "/home/brendan/AI-Shared/AntigravityProjects/tradovate dashbaord/start-tradovate-dashboard.sh"
```

This will:
- install dependencies if needed
- start the local sync helper
- start the Vite app
- try to open the app in your browser

Default app URL:
- <http://127.0.0.1:5173>

## Stop

Run:

```bash
bash "/home/brendan/AI-Shared/AntigravityProjects/tradovate dashbaord/stop-tradovate-dashboard.sh"
```

## Logs

- App log: `runtime-logs/vite.log`
- Helper log: `runtime-logs/sync-helper.log`
